# Universe Marketing Group

Static portfolio site for Universe Marketing Group.

## Deploy

This folder is ready to upload to GitHub and connect to Vercel.

Use these Vercel settings:

- Framework Preset: Other
- Build Command: leave empty
- Output Directory: leave empty

Public URLs after deployment:

- `https://universegroup.us/robots.txt`
- `https://universegroup.us/sitemap.xml`
